## 加载历史记忆

当用户说"加载之前的对话"、"继续上次的话题"、"恢复历史会话"等想要加载历史上下文时，使用此能力。

先读取 `.opencode/context.md` 文件获取可用的历史记忆列表，根据用户意图选择最匹配的记忆添加：

```
<!--ACTION:memory{"index":编号}-->
```

## 策略

- 用户明确说了编号或关键词 → 直接加载匹配的
- 用户只说"加载记忆" → 列出可用的记忆让用户选择（不加标签）
- 没有可用记忆 → 告知用户暂无历史记忆

## 规则

- ACTION 标签放在回复文本最后一行
- 不确定时列出选项让用户选，不要猜
